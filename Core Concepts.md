# Core Concepts

Traditional DeFi systems rely on:

- Passive staking yields  
- Isolated liquidity strategies  
- Single-layer yield generation  

CurveYield introduces a multi-layer yield architecture:

Base Yield + Strategy Yield + Liquidity Yield = Embedded Yield

## Tradeable Liquid Derivatives

Deposited positions can be represented as liquid, tradeable assets, allowing users to:

- Maintain liquidity  
- Exit positions efficiently  
- Use positions as collateral  
- Trade similarly to the underlying asset when deployed into LPs
