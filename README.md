# Introducing the CurveYield Protocol

### Introducing the CurveYield Protocol

CurveYield is a protocol that supports the Curve ecosystem by building integrations between Curve and other DeFi protocols.

It is designed as a broad integration layer for Curve, enabling new ways for Curve ecosystem assets to be used across DeFi. This includes vaults, lending markets, liquidity systems, yield-bearing tokens, and other infrastructure that improves the utility, liquidity, and yield potential of the Curve Ecosystem.

The long-term purpose of CurveYield is to support the growth of the Curve ecosystem—expanding Curve’s total value locked (TVL), increasing Curve’s protocol revenue, growing the Curve user base, and improving the stability of crvUSD to enable greater scale—while also providing users with ways to generate higher yield, reduce risk, and simplify complex strategies.
