# CurveYield crvUSD Ethereum Vault (cy-crvUSD)

CurveYield crvUSD on Ethereum is an IPOR Fusion vault that accepts crvUSD and issues cy-crvUSD vault shares.

The Ethereum vault was created through the deployed IPOR Fusion factory. It uses crvUSD as the underlying asset and is intended to support a curated set of Curve and StakeDAO yield routes after the final approved collateral and pool set is selected.

The vault has creator fee configuration and role transfer set up for CurveYield governance. Collateral-specific routes and oracle sources are not listed here as active strategy routes until they are approved and configured.
